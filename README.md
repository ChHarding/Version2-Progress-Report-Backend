# Version2-Progress-Report-Backend
1. Download the files.
2. Install Python version 3.12 or above.
3. Install pip (same version as Python)
4. Install pandas
5. Install Flask
6. Install openxyl
7. Run app.py
8. Navigate to http://127.0.0.1:4242
9. You will land on the home page.
